---

Witty Data UI components for modern apps

## Features

- {Switch} 
- {FormRow} 

## Install

```bash
yarn add witty-ui
```
OR
```bash
npm install witty-ui --save
```

## Usage

```jsx
import { Switch } = require('witty-ui');

render () {
    return (<Switch onLabel='Yes' offLabel='No' />)
}

```
